W tym pakiecie zostały umieszczone klasy odpowiedzialne za wykonywanie pewnych akcji. Wszystkie klasy z tego pakietu implementują interfejs {@link java.awt.event.ActionListener}. Klasy te połączone są z programem jako "słuchacze" pozycji z paska menu lub menu dostępnego pod prawym przyciskiem myszy. Wszystkie klasy otrzymują w konstruktorze obiekt klasy {@link modgraf.view.Editor}, co umożliwia im modyfikowanie grafów zarówno warstwy wizualnej jak i matematycznej.
