W tym pakiecie zostały umieszczone klasy odpowiedzialne za integrację warstwy wizualnej i matematycznej. Mechanizm integracji oparłem na zdarzeniach generowanych przez graficzną bibliotekę JGraphX. Każda zmiana wyglądu grafu powoduje wygenerowanie w kodzie zdarzenia. Niektóre z tych zmian wymagają modyfikacji warstwy matematycznej np. dodanie wierzchołka lub usunięcie krawędzi. Są też zmiany, które nie mają na nią wpływu np. zmiana położenia wierzchołka lub koloru jego wypełnienia. Klasy w tym pakiecie reagują tylko w momencie zmian wymagających modyfikacji warstwy matematycznej.
